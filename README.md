# FileSystemApp

## Описание

FileSystemApp — WPF приложение с архитектурой MVVM, моделирующее файловую систему и предоставляющее механизм рефлексии для динамической загрузки сборок.  
Позволяет создавать папки и файлы, перемещать и копировать их, а также загружать любую .NET-библиотеку, находить классы, реализующие интерфейс `IReflectable`, выбирать конструкторы и методы, вводить параметры и выполнять вызовы.

### Особенности

- Базовый абстрактный класс `FileSystemElement` с абстрактными свойствами `ElementType` и `Size`.
- Два наследника: `File` и `Folder`.
- Папки содержат список элементов, размер папки вычисляется автоматически.
- Статические методы `Move` и `Copy` с событиями уведомления.
- Полностью реализована MVVM архитектура.
- WPF интерфейс с `TreeView`.
- Вычисляемое свойство `Path` для отображения пути элемента.
- Отдельный модуль `FileSystemLib` со всеми моделями.
- Рефлексия: загрузка сборки, фильтр по интерфейсу `IReflectable`, отображение конструкторов и методов, ввод параметров, создание объекта и вызов метода.

## Архитектура

MVVM:

* Models (в отдельной библиотеке `FileSystemLib`)
* ViewModels
* Views


## Использование

### Главное окно

- Слева — дерево папок и файлов (тестовая структура создаётся автоматически).
- Выберите элемент — справа отобразятся его имя, путь, размер.
- Выберите целевую папку в нижнем дереве.
- Кнопки **Move** и **Copy** выполняют перемещение или копирование.
- Кнопка **Reflection** открывает окно рефлексии.

### Окно рефлексии

1. Введите полный путь к DLL (например, `C:\Projects\FileSystemLib\bin\Debug\net6.0\FileSystemLib.dll`).
2. Нажмите **Загрузить сборку**.
3. Выберите класс из списка (только классы, реализующие `IReflectable`).
4. При необходимости выберите конструктор — под ним появятся поля для ввода параметров.
5. Выберите метод — под ним появятся поля для ввода параметров.
6. Введите значения параметров (строки, числа и т.д.).
7. Нажмите **Выполнить** — будет создан объект и вызван метод, результат отобразится в сообщении.

## Пример класса для тестирования рефлексии

Добавьте в проект `FileSystemLib`:

```csharp
using FileSystemLib.Models;

namespace FileSystemLib.Additional
{
    public class Calculator : IReflectable
    {
        public int Add(int a, int b) => a + b;
        public double Multiply(double x, double y) => x * y;
        public string Greet(string name) => $"Hello, {name}!";
    }
}
